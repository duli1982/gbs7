Job Description Creator

You are an expert RPO recruiter and recruitment marketer. Write a compelling job description for a [ROLE_TITLE] position at [COMPANY_TYPE]. Requirements: - Include clear responsibilities and requirements - Use engaging, benefit-focused language - Specify required experience level - Add compensation range if provided - Keep length around 300-400 words - Avoid buzzwords and jargon Output format: - Role Summary - Key Responsibilities - Required Skills - Nice-to-Have Skills - Compensation & Benefits

Boolean Search Generator

Create advanced Boolean search strings for finding [TARGET_ROLE] candidates. Consider: - Alternative job titles and synonyms - Required skills and technologies - Experience levels - Location preferences - Company size/industry focus Provide 3 different search strategies: 1. Broad discovery search 2. Targeted skills-based search 3. Competitive intelligence search Format each as ready-to-use LinkedIn/Google search strings.

Interview Question Framework

Generate behavioral interview questions for a [ROLE_LEVEL] [JOB_TITLE] position. Create questions that assess: - Technical competency - Leadership potential - Cultural fit - Problem-solving approach - Communication skills For each question, provide: - The question text - What you're evaluating - Sample strong vs. weak responses - Follow-up questions Generate 8-10 questions total across different competency areas.
